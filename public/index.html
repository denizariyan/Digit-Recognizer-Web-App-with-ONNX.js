This demo is a digit recognizer made with the ONNX.js framework from Microsoft. Similar to tensorflow.js, ONNX.js is another framework to provide the capability of running machine learning models with JavaScript.

The advantage of using ONNX.js for this project is that no data is being sent to anywhere, everything happens on the user's computer using WebAssembly and WebGL to run on CPU or GPU respectively. Also, since everything is running on the user's local computer there is no need for powerful servers or concerns about privacy.

One of the disadvantages of making evaluations on the client-side is that the project depends on the user's computer being powerful enough to handle the model at a fast pace. This is not a problem in this case as the used MNIST model is relatively small and optimized making it easy on the client-side computer.

The model that is being used to make predictions is not perfect by any means. It is trained to be fast and easy to run, compromising the accuracy. This is merely a demo to show the capabilities of the ONNX.js and how easy it makes running machine learning models on the web. Run on a powerful computer with a large monitor for best experience.
